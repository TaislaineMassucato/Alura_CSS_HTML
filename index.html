Passos para HTML Básico


HTML é uma linguagem de Marcação(TAGS)

  1. Construindo a estrutura básica do HTML:

    Crie um documento HTML com as tags essenciais: <!DOCTYPE html>, <html>, <head>, e <body>. Adicione um <title> na seção <head> para nomear a página.

  2. Adicionando conteúdo ao HTML:

    Insira um cabeçalho com a tag <h1> e um parágrafo com a tag <p> no corpo do documento.

  3. Trabalhando com meta tags e título:

    Otimize o <head> adicionando uma meta tag para definir a codificação UTF-8 e altere o título para algo mais descritivo.

  4. Organizando conteúdo com tags HTML:

    Crie uma página com um título principal (<h1>), um subtítulo (<h2>), e um parágrafo explicativo (<p>).

  5. Adicionando imagens com acessibilidade:

    Use a tag <img> para inserir uma imagem, incluindo o atributo alt para melhorar a acessibilidade.

  6. Listando Itens em HTML:

    Adicione uma lista de itens com <ul> (não ordenada) ou <ol> (ordenada) e utilize <li> para cada item.

  7. Explorando o impacto do DOCTYPE no HTML:

    Crie uma página HTML com e sem a declaração <!DOCTYPE html> e compare os modos de renderização (Quirks Mode vs. Standards Mode).

  8. Automatizando a atualização da página com Live Server:

    Instale e use o "Live Server" no VS Code para ver alterações em tempo real ao editar o código HTML.
